### 사용법

0. 스크립트 추가<br>
UIManger 를 어딘가에다가 넣어둡니다.

1. 네임스페이스 포함
```cs
// 베이스 네임스페이스
using UI;

// 기본 네임스페이스
using UI.Manager;

// 세부적
using UI.Manager.Button; // 버튼 관련
using UI.Manager.Slider; // 슬라이더 관련
```
